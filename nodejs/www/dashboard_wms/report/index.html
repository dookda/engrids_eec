ที่มา : กรมการปกครอง กระทรวงมหาดไทย
การคาดการณ์ประชากรจังหวัดในเขตเศรษฐกิจพิเศษของสำนักงานคณะกรรมการนโยบายเขตเศรษฐกิจพิเศษภาคตะวันออก (EEC)
การคาดการณ์ประชากรแฝงจังหวัดในเขตเศรษฐกิจพิเศษของสำนักงานคณะกรรมการนโยบายเขตเศรษฐกิจพิเศษภาคตะวันออก (EEC)
อัตราการเติบโตของประชากรในเขตเมือง

ที่มา : แผนภาพรวมเพื่อการพัฒนาเขตพัฒนาพิเศษภาคตะวันออก, สำนักงานคณะกรรมการนโยบายเขตพัฒนาพิเศษภาคตะวันออก (สกพอ.), 2561

การคาดการณ์ประชากรของ 3 จังหวัดทั้งประชากรตามทะเบียนบ้านและประชากรแฝง จากผลกระทบของโรคโควิด-19

หมายเหตุ

  • คาดการณ์ตามเงื่อนไข คือ ตราการเพิ่มคิดตามอัตราเฉลี่ย 5 ปี (พ.ศ.2558-2562) ของแต่ละจังหวัด (อัตราการเพิ่มของจังหวัดชลบุรี : ระยอง : ฉะเชิงเทรา อยู่ที่: 1.86 : 1.73 : 0.70)
การคาดการณ์ความต้องการใช้น้ำแต่ละภาคส่วนในพื้นที่เขตพัฒนาพิเศษภาคตะวันออก (EEC)

ที่มา : สำนักงานทรัพยากรน้ำแห่งชาติ (2562)

การใช้น้ำอุตสาหกรรมในพื้นที่ EEC ในช่วงที่ผ่านมา (ล้านลบ.ม./ปี)

ที่มา : สถาบันน้ำและสิ่งแวดล้อมเพื่อความยั่งยืน สภาอุตสาหกรรมแห่งประเทศไทย (2562)

ที่มา : แผนปฏิบัติการการพัฒนาโครงสร้างพื้นฐาน รองรับการพัฒนาเขตพัฒนาพิเศษภาคตะวันออก, สำนักงานคณะกรรมการนโยบายเขตพัฒนาพิเศษภาคตะวันออก (สกพอ.), 2561

การคาดกาณ์ความต้องการใช้น้ำประปาในพื้นที่ EEC ในอนาคต (ล้านลบ.ม./ปี)

ที่มา : แผนปฏิบัติการการพัฒนาโครงสร้างพื้นฐาน รองรับการพัฒนาเขตพัฒนาพิเศษภาคตะวันออก, สำนักงานคณะกรรมการนโยบายเขตพัฒนาพิเศษภาคตะวันออก (สกพอ.), 2561

การคาดการณ์ความต้องการใช้พลังงานไฟฟ้าในเขตพื้นที่เศรษฐกิจพิเศษภาคตะวันออก (EEC)

ที่มา : แผนปฏิบัติการการพัฒนาโครงสร้างพื้นฐาน รองรับการพัฒนาเขตพัฒนาพิเศษภาคตะวันออก, สำนักงานคณะกรรมการนโยบายเขตพัฒนาพิเศษภาคตะวันออก (สกพอ.), 2561

การคาดการณ์ปริมาณน้ำเสียชุมชนในกรณีที่มีการพัฒนาพื้นที่เขตพัฒนาพิเศษ ภาคตะวันออก (EEC)

หมายเหตุ

  • จำนวนประชากรที่คาดการณ์ นำมาคูณกับอัตราการเกิดน้ำเสียของชุมชน 150 ลิตร/คน/วัน
  • จำนวนประชากรที่คาดการณ์เป็นการคาดการณ์ประชากรจังหวัดในเขตเศรษฐกิจพิเศษของสำนักงานคณะกรรมการนโยบายเขตเศรษฐกิจพิเศษภาคตะวันออก

ที่มา : ใช้ข้อมูลการคาดการณ์ประชากรจาก สำนักงานคณะกรรมการนโยบายเขตพัฒนาพิเศษภาคตะวันออก (สกพอ.), 2561 คูณกับอัตราการเกิดน้ำเสียของชุมชน ปีฐาน พ.ศ. 2563 ซึ่งเท่ากับ 150 ลิตรต่อคนต่อวัน (กรมควบคุมมลพิษ, อ้างใน ๒๕๕๓ สำนักงานสิ่งแวดล้อมภาคที่ 13 (ชลบุรี), 2563)

คำนวณคาดการณ์ปริมาณน้ำเสียชุมชน (ลิตร/คน/วัน)

  
ลิตร/วัน
การคาดการณ์ปริมาณขยะมูลฝอยในพื้นที่เขตเศรษฐกิจพิเศษภาคตะวันออก (EEC)

หมายเหตุ

  • ปริมาณขยะที่จะเกิดขึ้นในอนาคตคำนวณมาจาก จำนวนประชากรจากการคาดการณ์ (คน) คูณด้วยอัตราการเกิดขยะมูลฝอย ซึ่งเป็นค่าคงที่ 1.17 กิโลกรัมต่อวัน อ้างอิงจากกรมส่งเสริมคุณภาพสิ่งแวดล้อม กระทรวงทรัพยากรธรรมชาติและสิ่งแวดล้อม
  • จำนวนประชากรที่คาดการณ์เป็นการคาดการณ์ประชากรจังหวัดในเขตเศรษฐกิจพิเศษของสำนักงานคณะกรรมการนโยบายเขตเศรษฐกิจพิเศษภาคตะวันออก พ.ศ. 2564-2570

ที่มา : สำนักงานคณะกรรมการนโยบายเขตพัฒนาพิเศษภาคตะวันออก (สกพอ.), มีนาคม (2562)

คำนวณคาดการณ์ปริมาณปริมาณขยะมูลฝอย (กิโลกรัม/คน/วัน)

  
กิโลกรัมต่อวัน
การคาดการณ์ปริมาณของเสียที่เกิดขึ้นในพื้นที่เขตเศรษฐกิจพิเศษภาคตะวันออก

ที่มา : ศูนย์ความเป็นเลิศด้านการจัดการสารและของเสียอันตราย (ศสอ.) HSM (อ้างถึงใน ข่าวกรุงเทพธุรกิจ 2562)

การคาดการณ์ปริมาณน้ำดิบในพื้นที่เขตพัฒนาพิเศษภาคตะวันออก (EEC)

ที่มา : แผนปฏิบัติการการพัฒนาโครงสร้างพื้นฐาน รองรับการพัฒนาเขตพัฒนาพิเศษภาคตะวันออก, สำนักงานคณะกรรมการนโยบายเขตพัฒนาพิเศษภาคตะวันออก (สกพอ.), 2561

การพยากรณ์เศรษฐกิจในพื้นที่เขตพัฒนาพิเศษภาคตะวันออก (EEC)

ที่มา : แผนปฏิบัติการการพัฒนาโครงสร้างพื้นฐาน รองรับการพัฒนาเขตพัฒนาพิเศษภาคตะวันออก, สำนักงานคณะกรรมการนโยบายเขตพัฒนาพิเศษภาคตะวันออก (สกพอ.), 2561

ที่มา : รายงานการศึกษาวิเคราะห์ฐานข้อมูลแรงงานและประมาณการความต้องการแรงงาน โดย กระทรวงแรงงาน สำนักงานปลัดกระทรวง กองเศรษฐกิจการแรงงาน (2560)

การคาดการณ์ปริมาณการปล่อยก๊าซเรือนกระจกในกรณี BAU

หมายเหตุ : ktCO2eq หมายถึง พันตันคาร์บอนไดออกไซต์เทียบเท่า

ที่มา : แผนที่นำทางการลดก๊าซเรือนกระจกของประเทศไทย ปี พ.ศ.2564–2573, สำนักงานนโยบายและแผนทรัพยากรธรรมชาติและสิ่งแวดล้อม กระทรวงทรัพยากรธรรมชาติและสิ่งแวดล้อม

การคาดการณ์ประเภทการใช้ประโยชน์ที่ดินในพื้นที่เขตพัฒนาพิเศษภาคตะวันออก (EEC)
