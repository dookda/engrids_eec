ระบบรายงานสถานการณ์คุณภาพอากาศ

เป็นระบบที่รวบรวมข้อมูลจากสถานีตรวจวัดสภาพอากาศของกรมควบคุมมลพิษที่บริการข้อมูลเปิด API ประกอบด้วย ข้อมูลดัชนีคุณภาพอากาศ (AQI) ก๊าซโอโซน (O3) คาร์บอนมอนอกไซด์ (CO) ก๊าซไนโตรเจนไดออกไซด์ (NO2) ก๊าซซัลเฟอร์ไดออกไซด์ (SO2) ฝุ่นละอองขนาดไม่เกิน 2.5 ไมครอน (PM2.5) และขนาดไม่เกิน 10 ไมครอน (PM10) เพื่อรับรู้สถานการณ์คุณภาพอากาศในพื้นที่เขตพัฒนาพิเศษภาคตะวันออกและพื้นที่โดยรอบ โดยข้อมูลที่อยู่ในระบบสามารถสืบค้นย้อนหลัง 14 วัน
