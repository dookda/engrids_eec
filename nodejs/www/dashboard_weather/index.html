ระบบรายงานสถานการณ์ปริมาณน้ำผิวดิน

ระบบรายงานสถานการณ์ปริมาณน้ำผิวดินเป็นระบบที่รวบรวมข้อมูลจากสถานีตรวจวัดสภาพอากาศของกรมอุตุนิยมวิทยาที่บริการข้อมูลเปิด API ประกอบด้วยข้อมูลปริมาณน้ำฝน อุณหภูมิ ความชื้นสัมพัทธ์ ความกดอากาศ และความเร็วลม เพื่อรับรู้สถานการณ์น้ำที่จะลงสู่ผิวดินในพื้นที่เขตพัฒนาพิเศษภาคตะวันออกและพื้นที่โดยรอบ โดยข้อมูลที่อยู่ในระบบสามารถสืบค้นย้อนหลัง 14 วัน

ข้อมูลลักษณะสภาพอากาศ ณ ตำแหน่งผู้ใช้งาน

ข้อมูลลักษณะสภาพอากาศ วันที่ เวลา สถานี (จากกรมอุตุนิยมวิทยา)
